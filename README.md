# React App

## Notes

## Setup

Install babel to support react

```
npm i -D babel-core babel-loader@7 babel-preset-env babel-preset-react babel-preset-stage-2
```

```
npm i react react-dom
```

## VSCode

Configure eslint for React

```
npx eslint --init
```
